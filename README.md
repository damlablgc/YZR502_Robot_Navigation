### Navigasyonu Başlatma
Simülasyonu ve navigasyon düğümlerini başlatmak için paket klasörüne gidip dosyayı doğrudan çalıştırabilirsiniz:

```bash
export TURTLEBOT3_MODEL=burger
# Proje klasörüne gidin
cd ~/catkin_ws/src/yzr502_proje/ros_navigation
# Launch dosyasını tam yoluyla çalıştırın
roslaunch otonom_surus.launch
